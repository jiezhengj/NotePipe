# Development workflow

## 在 Obsidian 中重载插件

1. 打开 **Preferences**
2. 点击 **Community plugins**
3. 在 **Installed plugins** 中找到你的插件
4. 关闭开关禁用插件
5. 打开开关启用插件

## 文件变更时自动重载

[Hot-Reload](https://github.com/pjeby/hot-reload) 插件会在源码变更时自动重载你的插件。

详见 [论坛公告](https://forum.obsidian.md/t/plugin-release-for-developers-hot-reload-the-plugin-s-youre-developing/12185)。
